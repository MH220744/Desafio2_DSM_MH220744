##  Descripción

Este repositorio contiene el desarrollo del **Desafío 2** de la materia **Desarrollo de Software para Móviles (DSM)**.

La aplicación implementa funcionalidades relacionadas con la gestión de viajes, incluyendo navegación entre pantallas y manejo de datos dentro de la aplicación.

---

##  Funcionalidades principales

- Navegación entre pantallas
- Gestión de destinos
- Interfaz de usuario interactiva
- Validación de datos
